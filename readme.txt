Allows the shipping fee to be set for different locations

== Description ==

This plugin will allow the shipping fee to be set for different locations

== Installation ==

1. Ensure you have latest version of Woocommerce plugin installed
2. Unzip and upload contents of the plugin to your /wp-content/plugins/ directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Quantity selector and Product delete icon will appear on the checkout page of your store.
